### 프로젝트 실행 방법
1. `/signup` 경로를 입력해 SignUp 페이지로 들어간다.
2. 회원가입에 성공하면 자동으로 SignIn 페이지로 이동한다.
3. 이메일과 비밀번호를 입력해 로그인에 성공하면 자동으로 Todo 페이지로 이동한다.
4. 서비스를 이용한다.
